About

I am currently an Interactive Web Design and development student at Georgian College of Applied Arts and Technology in Barrie, Ontario, Canada.

Beginning from when I was a child I have been interested in Technology, specifically the internet. The internet is a beautiful place, this is because of designers. If there were no designers like myself with a passion for the Internet then life would be so very different, we would still be looking at plain text on a screen without design or functionality.

The internet has come a long way in a relatively short time and I am proud to be a part of that ever changing and developing world which not only changes lives with research and information but socially and entertaining as well.

I hope to have a long and interesting career working for a company that has a broad vision and wants to be a leader in the changes that the internet is bound see in the future. There is no feeling like seeing the changes that you make help and shape lives on the internet.

The Gear I use...

I currently use a Macbook Pro 15 inch (late 2013) with retina display running the latest Mac operating system. As well I am using a Virtual Machine (VM) to run Windows 10 Pro. I also run a Windows desktop running Windows 7, 8.1, 10 as well as Linux Ubuntu. I find working on many platforms and operating systems keeps things interesting also it keeps my knowledge up to date, so that way wherever I am I can get work done fast, effectively and efficiently.
